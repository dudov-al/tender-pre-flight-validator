# Pre-flight валидатор тендерной заявки

**Micro-SaaS для тендерщиков, который проверяет заявки (44-ФЗ/223-ФЗ) на ошибки и несоответствия ДО подачи на ЭТП.**

---

## О проекте

### Проблема
Отклонения заявок из-за формальностей/несоответствий → потерянная выручка, штрафы, потраченное время.

### Решение
Pre-flight чекер, который ловит 90% причин отклонений ДО подачи.

### Ценностное предложение
> «Не теряйте контракты из-за формальностей. Проверьте заявку перед подачей — бесплатно.»

## MVP Фокус

**Версия v1.0:** 44-ФЗ + услуги (самый крупный сегмент)

### P0 фичи (25 штук)

1. **Загрузка и анализ документов (8 фич)**
   - Drag & drop загрузка
   - Пакетная загрузка
   - Поддерживаемые форматы (PDF, DOCX, ZIP, SIG)
   - Лимит размера
   - Сканирование текста
   - Отчет по анонимности
   - Проверка наличия КЭП
   - Валидность сертификата

2. **Проверка подписей (3 фичи)**
   - Подписант vs Должность
   - Проверка доверенности
   - Проверка свежести доверенности

3. **Проверка обязательных документов (4 фичи)**
   - Чек-лист документов
   - Проверка лишних документов
   - Проверка свежести выписки из ЕГРЮЛ
   - Проверка приемлемости НМЦК

4. **Проверка Формы 2 (4 фичи)**
   - Проверка обязательных полей
   - Проверка страны происхождения
   - Проверка на расплывчатые формулировки
   - Проверка на противоречия в характеристиках

5. **Инфраструктура и дедлайны (3 фичи)**
   - Валидность сертификата КЭП
   - Отслеживание дедлайна
   - Технический тайм-аут

6. **Отчеты и чек-листы (5 фичи)**
   - Итоговая сводка
   - Детальный отчет по ошибкам
   - Критичность
   - Чек-лист "Перед загрузкой"
   - Чек-лист "После Формы 2"
   - Чек-лист "Перед подачей"
