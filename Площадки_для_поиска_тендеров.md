# Площадки для поиска тендеров

**Проект:** Pre-flight валидатор тендерной заявки
**Федеральные законы:** 44-ФЗ, 223-ФЗ
**Дата:** 2025-02-04

---

## Кратко

Список площадок и инструментов для поиска и мониторинга тендеров (44-ФЗ, 223-ФЗ). Сгруппированы по типу: официальные источники, коммерческие агрегаторы, аналитические платформы.

---

## 1. Официальные источники

### ЕИС (Единая информационная система)
- **URL:** zakupki.gov.ru
- **Официальная площадка** для 44-ФЗ и части 223-ФЗ
- **Особенности:**
  - Бесплатно
  - Обязательна для публикации всех 44-ФЗ закупок
  - Поиск и фильтры базовые
  - Может быть медленно, особенно в пиковые периоды

### Официальные ЭТП (электронные торговые площадки)
- **Roseltorg (Росэлторг)** — roseltorg.ru
- **Сбербанк-АСТ** — sberbank-ast.ru
- **РТСТендер** — rts-tender.ru
- **ЕЭТП** — eetp.ru
- **Газпромбанк** — gazprombank.ru
- **ММВБ** -> сейчас часть функций переехала
- **НИУ ВШЭ** -> для образовательных закупок

## 2. Коммерческие агрегаторы поиска + аналитика

### Контур.Закупки (Kontur)
- **URL:** zakupki.kontur.ru
- **Цены:** 16 200 ₽/год (Лайт) → 67 000 ₽/год (Про)
- **Особенности:**
  - Сильная аналитика
  - Уведомления по фильтрам
  - API-модуль (от 26 000 ₽)
  - Интеграция с другими сервисами Контура
- **Для кого:** Те, кто уже пользуется Контуром (бухгалтеры, юристы)

### Тендерплан
- **URL:** tenderplan.ru
- **Цены:** 11 500 / 19 500 / 35 000 ₽ (3/6/12 мес)
- **Особенности:**
  - Контроль изменений
  - Расчет рисков
  - Календарь
  - Прогнозы
- **Для кого:** SMB, средние компании

### Saby / СБИС
- **URL:** saby.ru
- **Экосистема:** Поиск + аналитика + контент/обучение + ЭДО
- **Особенности:**
  - Сильная интеграция с СБИС (ЭДО, бухгалтерия)
  - Статьи о причинах отклонений
  - Много контента для обучения
- **Для кого:** Те, кто уже использует СБИС как основную систему

### Seldon
- **URL:** seldongroup.ru
- **Цены:** По запросу (enterprise-паттерн)
- **Особенности:**
  - Enterprise-ориентированный продукт
  - Встроенный AI/аналитика
  - Консалтинг
- **Для кого:** Крупные компании, государственные организации

### Закупки360
- **URL:** startpack.ru (приложение)
- **Цены:** Бесплатно и навсегда (по заявлению)
- **Особенности:**
  - Давит ценой на базовый поиск
  - Базовый функционал бесплатно
- **Для кого:** Небольшие компании, начинающие тендерщики

### BiCoTender
- **URL:** bicotender.ru
- **Особенности:**
  - ТRM (управление отделом) — похоже на CRM для тендеров
  - Сопровождение и обучение
  - Отчет "воронка участий"
- **Для кого:** Тендерные отделы, которые хотят систематизировать процесс

## 3. Специализированные площадки

### Стартендер (StarTender)
- **URL:** star-tender.ru
- **Функция:**
  - Образовательный контент
  - Практика организации работы тендерного специалиста
  - Статьи о том, как заполнять Форму 2 и готовить заявки

### Roseltorg
- **URL:** roseltorg.ru
- **Функция:**
  - ЭТП (электронная торговая площадка)
  - База знаний (азбука закупок)
  - Единый реестр участников закупок (ЕРУЗ) — около 892 436 зарегистрированных участников

## 4. Общественные ресурсы и форумы

### forum.gov-zakupki.ru
- **Функция:** Форум тендерщиков
- **Ценность:**
  - Реальные случаи отклонений с цитатами
  - Вопросы-ответы по заполнению Формы 2
  - Обсуждение ошибок и проблем

### Клерк (Klerk)
- **URL:** klerk.ru
- **Разделы:** Бухгалтерия, ЭДО, финансы
- **Ценность:**
  - Блоги о сверках, первичке, выписках
  - Статьи об автоматизации бухгалтерии

## 5. Информационные источники (статистика и аналитика)

### Interfax.ru
- **Тип:** Бизнес-новости
- **Ценность:**
  - Статистика по рынку госзакупок (объемы, количество извещений)
  - Сводки Минфина по итогам мониторинга
- **Ключевые цифры:**
  - 2024: 3,58 млн контрактов на 11,98 трлн ₽ (44-ФЗ)
  - 2024: ~19 трлн ₽ (223-ФЗ)
  - II кв. 2025: 1,25 млн извещений на 6,74 трлн ₽

### Minfin.gov.ru
- **Тип:** Официальный сайт Минфина
- **Ценность:**
  - Сводные аналитические отчеты по госзакупкам
  - Статистика по участникам и контрактам

### Spectender.ru
- **Тип:** Аналитический портал по тендерам
- **Ценность:**
  - Обзоры и аналитика по рынку
  - Пересказ сводных отчетов Минфина

## 6. Рекомендации для нашего проекта

### Для поиска тендеров (участие в закупках)
**Если участвуем, чтобы понимать боли:**
- Использовать ЕИС (бесплатно)
- Добавить Контур.Закупки или Тендерплан для мониторинга и аналитики

### Для конкурентного анализа
**Что мониторить:**
- Функционал конкурентов (Контур, Тендерплан, Saby, Seldon)
- Цены (якорь willingness-to-pay)
- Промо-материалы (как они продают свои решения)
- Статьи и контент (что считается важным)

### Для маркетинга и продаж
**Где быть:**
- forum.gov-zakupki.ru — отвечать на вопросы, собирать живые формулировки отказов
- LinkedIn/Telegram-каналы о тендерах
- Вебинары и статьи на ресурсах конкурентов (учиться)

## 7. Интеграции для нашего продукта

### Для получения данных о закупках
- **ЕИС API** — открытый API для 44-ФЗ
- **ЭТП API** — для получения подробностей о закупках на конкретных площадках
- **Агрегаторы API** — Контур.Закупки (есть API-модуль от 26 000 ₽)

### Для проверки заявки
- **ЕИС/ЭТП** — статус закупки, время до дедлайна
- **КриптоПро API** — проверка КЭП
- **Госуслуги** — реестр сертификатов
